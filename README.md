# Calculator App using Tkinter

## 📌 Project Description
This is a simple and colorful calculator application built using Python Tkinter.  
The calculator performs basic arithmetic operations like:

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (/)
- Percentage (%)

It also includes:
- Clear button
- Backspace/Cancel button
- Modern dark-themed UI

---

- Python
- Tkinter GUI Library
